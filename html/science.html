Консультант Прядка Владислав Сергеевич

Сети играют важную роль в нашей жизни, их использование распространено везде. Следовательно, к проблемам безопасности сетей и удаленного взлома становится все больше внимания. Так, одними способов защиты сетей является анализ сетевых пакетов.

Анализатор сетевых пакетов (англ. Network Packet Analyzer) — это программное обеспечение, которое используется для анализа сетевого трафика. Оно отображает информацию о передаче данных по сети, а также позволяет отслеживать, какой источник отсылает данные другому узлу.

Целью данной курсовой работы является разработка анализатора сетевых пакетов на языке Python.

В рамках курсовой работы было поставлено несколько задач:

  • Фиксация требований, предьявленных к проекту.
  • Изучение основ и принципов разработки приложений на языке Python.
  • Анализ существующих методов мониторинга сетей, реализация и тестирование анализатора сетевыx пакетов.
  • Реализация и тестирование анализатора сетевых пакетов

    • В коде работы были зафиксированы требования, предьявленные к проекту:

  • Анализатор сетевы пакетов должен быть написан на языке программирования Python.
  • Программа должна анализир овать порт, которь ввел клиент.
  • Сервис должен предоставлять понятный интер фейс для клиента.
  • Программа должна применять фильтр для анализа сетевых пакетов.
  • В программе должна быть возможность завершения программы.


    • Анализатор сетевых пакетов - это программа, которая захватьвает сетевые пакеты и позволяет анализировать данные, передаваемые по сети.

    Работа анализатор сетевых пакетов состоит из следующих шагов

  • В первую очередь - это захват сетевого пакета. Программа получаетпо сети пакеты, которые передаются от сетевого адаптера.
  • Далее проискодит обработка пакета с помощью декодера. Он анализирует заголовки пакета, распознает тип протокола и восстанавливает полезную нагрузку пакета.
  • После происходит анализ пакетов
  • Отображение пакетов
    		•

    Структура сервиса представляет собой четыре метода: analyze_port(), analyze_tcp(), analyze_http(), main().
    Первые три метода позволяют выполнить захват и анализ пакетов.
    Метод main() представляет собой пользовательский интерфейс, который позволяет пользователю легко и быстро просматривать информацию об анализируемых пакетах.

    Основные результаты работы

    Для тестирования работы использовалась консольная строка. На экране вы можете видеть вывод интерфейса для пользователя, вывод данных по введенному пользователем порту или выбору анализа. После программа выводит углубленные данные (различную информацию о сетевых пакетах, IP-адрес отправителя, IP-адрес получателя, протокол, длина пакета и другие параметр). А также совершает проверку на невнимательность.

Он помогает обеспечивать безопасность сети, оптимизировать ее рабол управлять трафиком и решать другие задачи.
