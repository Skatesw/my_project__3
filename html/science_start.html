Научная деятельность и/или интересы в области математики/программирования

В данной курсовой работе была разработан анализатор сетевых пакетов на языке Python с использованием библиотеки Scapy, который является мощным инструментом для анализа данных, передаваемые по сети, и отладки сетевого трафика. Были выполнены поставленные задачи. Изучены основы языка программирования Python, изучены принципы разработки приложений на языке Python, выполнен анализ существующих методов мониторинга сетей, разработан и протестирован анализатор сетевых пакетов.
